revealjs

Page en chantier - à raffiner avec ce diaporama stratégique

Ce raccourci-code mettra en forme le markdown entouré pour le restituer avec reveal.js sur la runtime (côté-client)

Pour en savoir plus, regardez le repo github revealjs.

Usage

revealjs peut utiliser les paramètres nommés suivants :

  • theme
  • transition
  • controls
  • progress
  • history
  • center
Important
Même si le contenu joint est du markdown, utilisez le < raccourci-code au lieu de la notation %
